# Macro Strategy Research Skill

This skill provides an interface to the Macro Strategy Research system.

## Tools

The system provides executable scripts located at `f:\Thomas\QuantSystem\.trae\skills\assets-score\scripts\`.

### 1. Prepare Context

Extract text from PDF reports and get scoring prompts.

**Usage:**

Run the preparation script. It will output the context (summary and prompts) to `macro_context.json` in the current directory.

```bash
python f:\Thomas\QuantSystem\.trae\skills\assets-score\scripts\prepare.py
# Optional: Specify date
# python f:\Thomas\QuantSystem\.trae\skills\assets-score\scripts\prepare.py --date 2026-02-04
```

After running, read `macro_context.json` to get the `summary_text` and `prompts`.

### 2. Archive Results

Save the scoring results generated by the Agent.

**Usage:**

1. Save your generated scoring results to a JSON file (e.g., `scores.json`).
2. Run the archive script:

```bash
python f:\Thomas\QuantSystem\.trae\skills\assets-score\scripts\archive.py scores.json
# Optional: Specify date
# python f:\Thomas\QuantSystem\.trae\skills\assets-score\scripts\archive.py scores.json --date 2026-02-04
```

## Workflow

1.  **Preparation**: Run `prepare.py`. Read `macro_context.json` to get `summary_text` and `prompts`.
2.  **Reasoning**: The Agent (You) generates scores for each style using the `summary_text` and `prompts`.
    *   **Step 2.1 (Base Scores)**: Generate **Conservative** and **Aggressive** scores independently.
        *   Use `prompts['conservative']` and `prompts['aggressive']` as system prompts.
    *   **Step 2.2 (Dialectical Score)**: Generate **Dialectical** scores by synthesizing the previous results.
        *   **Input**: `summary_text` + `Conservative Result` + `Aggressive Result`.
        *   **System Prompt**: `prompts['dialectical']`.
        *   **Logic**: Critically analyze the conflict between conservative and aggressive views to form a balanced judgement.
3.  **Archiving**: Save results to `scores.json`. Run `archive.py scores.json` to save to DB/Excel.

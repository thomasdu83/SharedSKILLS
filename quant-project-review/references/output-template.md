若为推断，标注：`初始目标为推断版本（置信度：中）`

**当前阶段**  
`idea / research / candidate / production / monitor_only`

**总体结论**  
`On Track / Slightly Drifting / Clearly Drifting`

**粗粒度完成度**  
使用区间表达，例如：`约 40%-50%`

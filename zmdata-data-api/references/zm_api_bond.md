
# ZM债券数据接口

## 可转债列表(cb_list)

可转债列表数据，包括可转债代码，可转债名称

**请求参数**

无

**请求响应**

| 字段    | 字段说明     | 类型   |
|---------|-------------|--------|
| cb_code | 可转债代码   | str    |
| cb_name | 可转债名称   | str    |


## 可转债行情(cb_daily_market_data)

可转债日行情数据，包括交易日期、开盘价、最高价、最低价、收盘价、成交量

**请求参数**

| 字段        | 字段说明   | 类型   | 是否必填 | 备注                                                          |
|-------------|-----------|--------|----------|---------------------------------------------------------------|
| cb_code     | 可转债编码 | str    | 是       | 例如：上交所的浦发转债：sh118054，深交所的齐翔转债：sz128127   |
| start_date  | 开始日期   | date   | 否       | 格式为YYYY-MM-DD，默认为转债上市日期                           |
| end_date    | 结束日期   | date   | 否       | 格式为YYYY-MM-DD，默认为当前日期                               |

**请求响应**

| 字段        | 字段说明   | 类型   |
|-------------|-----------|--------|
| cb_code     | 可转债编码 | str    |
| trading_day | 交易日期   | date   |
| open_price  | 开盘价     | float  |
| high_price  | 最高价     | float  |
| low_price   | 最低价     | float  |
| close_price | 收盘价     | float  |
| volume      | 成交量     | float  |
| turnover    | 成交额     | float  |

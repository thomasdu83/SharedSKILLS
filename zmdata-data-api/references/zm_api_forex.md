
# ZM外汇数据接口


## 获取汇率历史数据(forex_hist_em)
 
### 功能描述
获取指定货币对的历史汇率数据

### 请求参数 

| 参数名 | 类型 | 必填 | 说明 | 
|--------|------|------|------| 
| symbol | str | 否 | 汇率代码，默认为USDCNH（美元兑离岸人民币） | 

### 响应字段 

| 字段名 | 类型 | 说明 | 
|--------|------|------| 
| 日期 | str | 交易日期，格式：YYYY-MM-DD | 
| 代码 | str | 汇率代码 | 
| 名称 | str | 货币对名称 | 
| 今开 | float | 开盘汇率 | 
| 最新价 | float | 收盘汇率 | 
| 最高 | float | 最高汇率 | 
| 最低 | float | 最低汇率 | 
| 振幅 | float | 涨跌幅振幅 |

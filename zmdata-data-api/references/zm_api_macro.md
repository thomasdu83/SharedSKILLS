
# ZM宏观数据接口

## 获取美联储利率数据(macro_bank_usa_interest_rate)
 
### 功能描述
获取美联储利率历史数据，数据区间从19820927至今

### 请求参数 

无

### 响应字段 

| 字段名 | 类型 | 说明 | 
|--------|------|------| 
| 商品 | str | 数据类型描述 | 
| 日期 | str | 利率调整日期，格式：YYYY-MM-DD | 
| 今值 | float | 当期利率值 | 
| 预测值 | float | 市场预测利率值 | 
| 前值 | float | 上期利率值 | 

---

## 获取美国非农就业数据(macro_usa_non_farm)
 
### 功能描述
获取美国非农就业人口历史数据，数据区间从19700102至今

### 请求参数 

无

### 响应字段 

| 字段名 | 类型 | 说明 | 
|--------|------|------| 
| 商品 | str | 数据类型描述 | 
| 日期 | str | 数据发布日期，格式：YYYY-MM-DD | 
| 今值 | float | 当期非农就业人口变动数 | 
| 预测值 | float | 市场预测非农就业人口变动数 | 
| 前值 | float | 上期非农就业人口变动数 | 

---

# ZM现货数据接口

## 获取上海黄金交易所黄金价格数据(spot_hist_sge)

### 功能描述

获取上海黄金交易所指定品种的历史价格数据

### 请求参数


| 参数名    | 类型  | 必填  | 说明                                                                                                                                                                                                                    |
| ------ | --- | --- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| symbol | str | 否   | 品种代码，可选值：{'Au99.99', 'Au99.95', 'Au100g', 'Pt99.95', 'Ag(T+D)', 'Au(T+D)', 'mAu(T+D)', 'Au(T+N1)', 'Au(T+N2)', 'Ag99.99', 'iAu99.99', 'Au99.5', 'iAu100g', 'iAu99.5', 'PGC30g', 'NYAuTN06', 'NYAuTN12'}；默认为 'Au99.99' |


### 响应字段


| 字段名   | 类型    | 说明                 |
| ----- | ----- | ------------------ |
| date  | str   | 交易日期，格式：YYYY-MM-DD |
| open  | float | 开盘价                |
| close | float | 收盘价                |
| high  | float | 最高价                |
| low   | float | 最低价                |

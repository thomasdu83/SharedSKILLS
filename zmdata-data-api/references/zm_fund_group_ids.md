# ZM基金分组ID定义

## 文档说明
本文档列出常用基金分组的ID映射关系，包括zm样本库和市场跟踪池两大体系。
适用于查询市场跟踪池所有标签、赛道标签、基金池 `label_id` 映射。

**术语说明**：
- **基金分组 / Fund Universe**：也称为样本组、基金池、基金标签
- **zm样本库**：完整的公募/私募分层体系（按策略+波动率分类）
- **市场跟踪池**：精选代表产品（按具体策略赛道分类）

**使用说明**：
- 所有分组都使用 `label_id`（整数）标识
- `get_fund_in_label(label_id)` - 通用接口，支持所有样本分组空间查询
- 本文档仅列出常用分组

## 相关 API

### get_fund_in_label(label_id_list)
**功能**：获取指定分组(同时查询多个分组)的基金列表（通用接口）
**优点**：可精确查询具体赛道（如500指增）

---

## 查询指引

### 如何查询基金分组？
**场景：查询市场跟踪池分组**
- 使用 `get_fund_in_label(label_id_list)`
- 示例：查询500指增赛道 → `label_id_list=[42201002]`

### 注意事项

- label_id 为整数类型，传参时使用整数而非字符串
- 上层节点包含所有下层节点的基金（如 label_id `20201` 包含所有波动率子分类）

# ZM常用指数ID定义

## 文档说明
本文档列出常用指数的ID映射关系，包括至明朝阳全景系列指数和市场公开指数。

**使用说明**：
- 本文档仅列出高频使用的指数
- benchID 为整数类型，传参时使用整数而非字符串
- index_code 为字符串类型（用于市场公开指数）

## 风险因子收益数据

以下 `benchID` 是 `pm_risk_factor_exposure` 服务端用于计算子基金因子暴露的因子收益序列。优先用客户端封装 `risk_factor_returns` 查询；也可用 `index_price` / `get_index_price` 按 `bench_id_list` 查询。取响应中的 `Yield` 作为回归或滚动相关性计算使用的因子收益。`BenchName` 来自服务端列名映射使用的 `[DDerive]..BenchMarkInfo`，并已去除首尾空格；`pm_risk_factor_exposure` 响应字段为最终返回给用户的字段名。

常用调用：

```python
import zmdata as api

# 股票 Barra 因子收益，对应 strategy_type=1
stock_barra = api.risk_factor_returns(strategy_type=1, start_date="2024-01-01", end_date="2024-12-31")

# CTA 策略因子收益，对应 strategy_type=5
cta = api.risk_factor_returns(factor_set="cta", start_date="2024-01-01", end_date="2024-12-31")
```

## 相关 API

- `get_index_price(bench_id_list)` - 使用 benchID 获取指数价格
- `get_index_price(index_code_list)` - 使用 index_code 获取指数价格

## 查询指引

### 如何在 `get_index_price` 接口中使用指数ID？

1. **至明朝阳全景系列指数**：
   - 只能使用 `bench_id_list` 参数
   - 从上表中查找对应的 benchID（整数）

2. **市场公开指数**：
   - 可使用 `bench_id_list` 参数（benchID 整数）
   - 或使用 `index_code_list` 参数（index_code 字符串）

### 注意事项
- benchID 为整数类型，传参时使用整数而非字符串
- index_code 为字符串类型，格式为"代码"（如"000300"）
- 如需查询更多指数，请联系数据团队
